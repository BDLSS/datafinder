Accessibility

This is the accessibility statement for the Oxford University Libraries website. It covers the main site at https://datafinder-d2v.bodleian.ox.ac.uk/home/ and . Other sites which are linked from the top level are not covered by this statement. Work is continuing to further improve the accessibility and usability of the site. If you have any questions or comments about the accessibility of this site, please contact us

Navigation aids

The site uses a consistent navigation system visually located at the top of the page.

Tables

Tables are used to display tabular information only T he content of the page is contained in semantically labelled 'div elements.

Images

Images include descriptive ALT attributes. Purely decorative graphics would include null ALT attributes.

Visual design

This site uses cascading style sheets for visual layout. If your browser or browsing device does not support style sheets at all, the content of each page is still readable. The website uses a layout scheme that allows the scale of the page to be increased by increasing the text size in visual browsers for the visually-impaired.

See also the central Oxford University's Web Accessibility Policy
